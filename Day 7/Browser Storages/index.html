Browser Storages

Local Storage

  • Data stored in localStorage remain persistant, until and unless we/user delete that data.
  • Each domain has its own localStorage
  • Generally localStorage storage capacity is limited to 5mb, but it may vary browser to browser.

Session Storage

  • Data stored in Session Storage is cleared when window/tab is closed or browser is closed
  • Each domain has its own session Storage
  • Generally Session storage capacity is limited to 5mb, but it may vary browser to browser.

Cookie Storage

  • Data for cookie storage has expiry. if there is no expiry data can be stored for longer period of time
  • Each domain has its own cookie storage
  • Cookie storage has capacity of storing data of 4kb only, becase cookie is sent everytime when a request is sent from client/browser to server, to reduce the request header load 4kb space for cookie is standarized for all browsers.
